# 粒子交互项目

这是一个基于 Three.js 的 3D 粒子交互演示项目。

## 功能
- **粒子系统**: 2000 个悬浮粒子，随鼠标移动产生斥力交互。
- **图片上传**: 点击"添加图片"上传本地图片，图片将作为 3D 平面显示。
- **模型加载**: 点击"添加模型"上传 .glb/.gltf 文件。
- **交互控制**:
    - **移动/缩放/旋转**: 点击物体选中，使用快捷键 T (移动), R (旋转), S (缩放) 切换模式。
    - **删除**: 选中物体后按 Delete 或 Backspace 删除。
    - **视角**: 左键旋转，右键平移，滚轮缩放。

## 运行方法
由于使用了 ES Modules 和纹理加载，需要通过本地服务器运行。

### Python (推荐)
在当前目录下运行：
```bash
python -m http.server 8000
```
然后访问: [http://localhost:8000](http://localhost:8000)

### Node.js (http-server)
```bash
npx http-server .
```
